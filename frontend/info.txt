4.2. Integració amb el backend

    A cavalls-frontend, faràs crides a l’API del backend. P. ex. en un component React:

useEffect(() => {
  fetch('http://localhost:3001/anuncis')
    .then((res) => res.json())
    .then((data) => setAnuncis(data));
}, []);

Assegura’t que el backend està escoltant a un altre port (p. ex. 3001) per evitar conflictes.

Si tens problemes de CORS, instal·la cors al backend:

npm install cors

I afegeix al app.js:

const cors = require('cors');
app.use(cors());

Així React pot fer crides a l’API sense bloquejos.
